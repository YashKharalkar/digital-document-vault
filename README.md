# Digital Document Vault

A secure document management platform built using the MERN Stack. Users can upload, organize, manage, and share documents through a clean and responsive interface.

## Features

* User Authentication using JWT
* Document Upload and Management
* Folder Creation and Organization
* Document Search
* Public Shareable Links
* MongoDB Atlas Integration
* Docker Containerization
* Docker Compose Setup
* GitHub Actions CI Pipeline

## Tech Stack

### Frontend

* React.js
* React Router
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Authentication

* JWT
* bcryptjs

### DevOps

* Docker
* Docker Compose
* GitHub Actions

## Getting Started

### Clone Repository

git clone https://github.com/YashKharalkar/digital-document-vault.git

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

## Docker Setup

docker compose up --build

## CI Pipeline

GitHub Actions automatically:

* Installs dependencies
* Builds the frontend
* Validates project build status
